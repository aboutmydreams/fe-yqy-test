

+ 启动项目

```bash
yarn start
```

+ 项目打包

```
yarn build
```

## Path Structure

```
src
|- components // 可复用组件
|- pages // 由页面分开各自组件
|  |- homepage
|     |- EditAndTextMessage
|        |- AppRouter.jsx // 出口文件 容器组件
|        |- EditAndTextMessageUI.jsx // UI组件
|        |- style.css
|  |- login
|- router // 路由
|- App.js
|- index.js // entry
```
